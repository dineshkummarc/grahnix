Prototype.js Collaboration Tool

Hello!

This is a collaborative tool intended to allow people to share updates to prototype.js scripts and discuss or explain their changes. When you log in, you'll be presented with a list of available "rooms." These are like chat rooms or (actually..) class rooms. Each room has a shared prototype.js canvas and a shared code base. Users can edit code and share their revisions with others.

One last thing, this relies heavily on web sockets, so you'll need a browser that knows what to do with them. I tested this in Chrome developer build, so your mileage, etc.

Things to Know

To test your project, click "run." If you have any errors in your code, we will tell you then. When you're happy the code, click "share" to notify everyone in the room that new code is available. In the discussion section to your right, other people's code changes are available. You can choose any of their revisions by clicking the link provided.
